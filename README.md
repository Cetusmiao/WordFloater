# WordFloater - 单词悬浮窗 🪟📖

悬浮歌词风格的桌面英语单词背诵工具，支持刷词和做题两种模式。
> 让背单词像看歌词一样自然。

## 项目简介

WordFloater 是一款悬浮歌词风格的桌面单词背诵工具。灵感来源于日常工作中的碎片时间——上班时很难专门抽出大段时间背单词，但如果有一个小窗像音乐歌词一样悬浮在屏幕角落，每隔几秒闪过一个单词，不知不觉就能记住不少。

所以，WordFloater 诞生了：它安静地浮在桌面最上层，不抢注意力，却能在你瞟一眼屏幕的瞬间完成一次单词记忆。刷腻了还能切到做题模式，用选择题检验成果，答错的单词自动归入生词本，通过熟练度系统反复巩固，直到真正掌握。

整个项目全程采用 **Vibe Coding** 方式开发，由 **Xiaomi MiMo v2.5 Pro** 驱动完成。从第一行代码到第一个可用版本，共消耗约 **39 万 tokens**。没有手写一行界面代码，没有手动调试一个布局参数——纯粹通过自然语言描述需求，让 AI 理解意图、生成代码、修复 Bug、迭代功能。这本身也是一次关于「AI 编程能做到什么」的实验。
## 功能特性

### 刷词模式
- 每隔设定时间自动刷新显示单词和释义
- 单词/释义左右排版，超长文本自动滚动
- 点击任意一行可将单词加入生词本

### 做题模式
- 随机出题：中选英 / 英选中混合
- ABCD 四个选项，颜色反馈（绿色正确 / 红色错误）
- 答错的单词自动加入生词本

### 单词本系统
- **内置单词本**：程序自带单词表，可在设置中增删改
- **生词本**：自动收集刷词点击和做题答错的单词，带熟练度系统
- **外部辞书**：支持导入自定义 CSV 文件（每两列为一组：单词 + 释义）

### 熟练度系统
- 新加入的单词熟练度为 3（最高）
- 生词本做题：答对 -1，答错 +1
- 熟练度降为 0 时自动删除（已掌握）

## 快速开始

### 直接运行
```bash
pip install PyQt5
python run.py
```

### 打包为便携式 exe
```bash
pip install PyQt5 pyinstaller
python build.py
```
打包产物在 `dist/WordFloater.exe`，可单独拷贝运行。

## 外部辞书格式

CSV 文件，每两列为一组（单词 + 释义），一行可有多组：

```csv
abandon,放弃,abstract,抽象的
academic,学术的,accelerate,加速
```

支持编码：UTF-8、GBK、GB2312、GB18030。

## 内置数据格式 (builtin.csv)

5 列结构：

| 列 | 说明 |
|---|---|
| 第1列 | 内置单词本 - 单词 |
| 第2列 | 内置单词本 - 释义 |
| 第3列 | 生词本 - 单词 |
| 第4列 | 生词本 - 释义 |
| 第5列 | 生词本 - 熟练度（数字） |

删除任意一行后自动挪位补齐，不留空行。
